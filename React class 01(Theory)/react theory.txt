REACT hamary pas isi liye ayi the kiu ky iss my ham HTML,JS ka code aik hi file my kar rahy hoty han jisy JSX file bhe bolty han (JAVASCRIPT XML). REACT was introduced by FACEBOOK.ANGULAR(framework) was developed by GOOGLE.

2) Jesy ky ap ny name suna hoga react angular vue nextJs halankay yah frameworks han, lekin React aik library han jo still itni powerfull library hay ky isy frameworks ky sath compare kiya jata hay    
3) LIBRARY VS FRAMEWORK
Library hamary pas specific tools hoty han kuch prewritten code hota yah kuch functions hoty han jin ka ham use karty han apnay code my.
Framework 1 structure hota hay 1 blue print hota hay yah 1 design hota hay jis ka ham use karty han iss ky apnay kuch rules hoty han iss ka apna kuch structure hota hay jis ko ham follow karty han  
Basic example: Library vs Framework
